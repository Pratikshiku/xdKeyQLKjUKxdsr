## Campus Online Store

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3f5f0f20932c48e9bd494ae7da4cad95/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, shops, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Website Introduction
5. Contact Us
6. Announcements
7. Shops
8. Products
9. Personal Center
10. Delivery Address
11. Product Collection
12. Product Orders
13. Shopping Cart
14. Change Password
15. Personal Information
16. Announcement Management
17. Product Management
18. Product Reviews
19. Product Collection
20. Product Orders




**Shop Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Management
6. Product Management
7. Product Review Management
8. Product Order Management




**Administrator Functions:**
1. Login
2. Personal Center
3. Announcement Type Management
4. Member Level Type Management
5. Shop Credit Type Management
6. Product Type Management
7. Announcement Management
8. Shop Management
9. Product Management
10. Product Review Management
11. Product Collection Management
12. Product Order Management
13. Single Page Data Management
14. User Management
15. Carousel Image Management
